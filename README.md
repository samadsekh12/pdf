# PDFSuite Pro

A modern web application for handling PDF files and images with various tools and features.

## Features

- PDF Merging
- PDF Splitting
- Image Resizing
- Format Conversion
- AI-powered Tools

## Setup

1. Clone this repository
2. Open `index.html` in your browser
3. For development, use a local server
